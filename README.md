# Разработка модели для предсказания параметров симуляций (MD) по геометрии самоорганизованных частиц с использованием глубокого обучения для анализа пространственных конфигураций в коллоидных системах
Группа БМТ1-23М 

Нафикова Р.Р. · Жуков К.П. · Кохановская А.В. · Походяев О.И. · Бондарева А.А.  

Руководитель: Мошкова А.А.
## Задача

По равновесной пространственной конфигурации самоорганизованных коллоидных частиц под конфайнментом восстановить управляющие параметры симуляции — высоту ограничения *h* и дипольный момент *p*.

## Данные

8 757 равновесных конфигураций из MD-симуляций в HOOMD-blue (N = 10⁴ частиц, плотности ρ = 0.30–0.90).

## Модели

| Модель | R² для *h* | R² для *p* |
|---|---|---|
| Гребневая регрессия (KRR, RBF) | 0.904 | 0.971 |
| Случайный лес | 0.909 | 0.984 |
| **PhysicsGNN** | **0.996** | **0.997** |
